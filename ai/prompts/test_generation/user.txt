Тема теста: {topic}

Сгенерируй тест строго в JSON-формате:
{
  "topic": "...",
  "questions": [
    {
      "text": "...",
      "options": ["...", "...", "..."],
      "correct_option_index": 0
    }
  ]
}

Требования:
- ровно {questions_count} вопросов
- ровно {options_count} вариантов ответа у каждого вопроса
- ровно один правильный вариант в каждом вопросе
- correct_option_index должен указывать на правильный вариант
- текст вариантов не должен повторяться внутри одного вопроса
